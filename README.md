# NitiSetu: Policy Intelligence Platform

An AI-powered dashboard for Niti Aayog to analyze state-wise Aadhar authentication trends, predict future coverage, and generate policy recommendations.

## Tech Stack
- **Frontend:** React, Vite, TailwindCSS
- **Backend:** FastAPI, Python, MongoDB
- **AI:** Google Gemini 1.5 Flash

## Setup
1. `cd backend` -> `pip install -r requirements.txt` -> `python main.py`
2. `cd frontend` -> `npm install` -> `npm run dev`
